Piekna i prosta strona do wysyłania requesta i odbierania danych


Podaj Typ Requestu:
Create -> Post
Read -> Get
Update -> Put
Delete -> Delete


Wybierz scierzke API: Zrobilem tylko jeden kontroler, wiec tylko jedna scierzka, analogicznie mógł bym zrobić wiele więcej.


Reszta scierzki (np. id):
GET -> puste pole = wszyscy ludzie : id = osoba o danym id
POST -> id,pole1,pole2 np: 12,rafal,dzierzbinski
PUT -> id,numerpola,nowatresc np: 12,1,lukasz (numerpola > 0)
DELETE -> id usuw urzytkownika o danym id np: 1
